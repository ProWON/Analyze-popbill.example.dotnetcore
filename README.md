# popbill.example.netcore

================================
팝빌 API Example for ASP.NET Core
================================

* install .NET CLI by
::
    $ dotnet add package Popbill --version 1.0.4


연동 API 목차
------------------------------
* 전자세금계산서 API
* 전자명세서 API
* 현금영수증 API
* 문자메시지 API
* 팩스 API
* 카카오톡 API
* 홈택스(전자세금계산서) 연동 API
* 홈택스(현금영수증) 연동 API
* 휴폐업조회 API
